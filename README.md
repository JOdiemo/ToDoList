# 📖 [To Do List] <a name="about-project"></a>

**[To Do Website]** "To Do list" is a simple website that helps an individual organise daily chores by adding, removing or marking as completed errands set for theday. By building this application, manages data using JavaScripdt and is interactive.

## 🛠 Built With <a name="built-with"></a>
- HTML
- CSS
- JavaScript

### Key Features <a name="key-features"></a>
- Adding a task
- Deleting a task
- Completting a task

## 💻 Getting Started <a name="getting-started"></a>

To get a local copy up and running, follow these steps.

### Prerequisites

In order to run this project you need:
-A Computer (MAC or PC)
-code editor (VSCode,Atom etc...)
-A browser (Chrome,Mozilla,Safari etc...)
-Version Control System (Git and Github)

### Setup

Clone this repository to your desired folder:
cd my-folder
git clone https://github.com/JOdiemo/ToDoList.git


cd ToDoList

### Install
npm install

### Usage
Install Live Server on VS Code and open index.html on your browser
